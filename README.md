# Netflix Clone 🎬

A responsive Netflix clone built with HTML and CSS that replicates the core UI/UX of Netflix's streaming platform.

## Features ✨
- **Hero Section** with email signup form
- **Trending Now** carousel with hover effects
- **"More Reasons to Join"** benefit cards with SVG icons
- **FAQ Accordion** with smooth animations
- **Responsive Design** works on mobile/tablet/desktop
- **Dark Mode UI** matching Netflix's color scheme
- **Interactive Elements** (hover states, form validation)
